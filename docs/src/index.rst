======================
SKA ODA Documentation
======================

The Observatory Science Operations Data Archive (ODA) is the primary storage for
entities involved in the OSO subsystems (for example, Scheduling Blocks or Execution Blocks).

For instructions on developing the ODA, see the `README <https://gitlab.com/ska-telescope/db/ska-db-oda/-/blob/main/README.md>`_
